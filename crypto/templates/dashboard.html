Visual Cryptography


Final Year Project - Oreoluwa Togun

Visual Cryptography is a cryptographic technique that
allows images or text to be encrypted in such a way that
the decryption can be performed directly by the human vi-
sual system, without the need for complex computations or
algorithms.

This technique involves dividing an image into multiple shares
or transparencies, which are then combined to reveal the orig-
nal image. e.g For this project we accepted a users share image
during login which was combined with the share image created by
the visual system in order to reveal the original image. If
the original image was created, after merge (validated) the user
gets authorized, else He/She is an unauthorized user.
